$

$ is used as a hexadecimal marker, so if you have $7FFE in your code it is the same as the constant 32766. This can improve readability.


print $2a
prints 42

*

Binary operator, multiplies two numbers


print 3*4
will display 12

+

Binary operator, adds two numbers


print 3+4
will display 7

-

Binary operator, subtracts two numbers


print 3-4
will display -1

/

Binary operator, divides two numbers and returns a floating point result.


print 11 / 4
will display 2.75

<

Compares two strings or numbers. Returns -1 if the first is less than the second, 0 otherwise.


if a < 1 then print "a is less than 1"

<<

A binary operator, shifts the left side left by the value in the right side. This is the same as multiplying by a power of 2 but it's much quicker


print 16 << 3
would print 128

<=

Compares two strings or numbers. Returns -1 if the first is less than or equal to the second, 0 otherwise.


if a <= 1 then print "a is less than or equal to 1"

<>

Compares two strings or numbers. Returns -1 if not equal, 0 if equal


if a <> 1 then print "a is not equal to 1"

=

Compares two strings or numbers. Returns -1 if equal, 0 if not equal


if a = 1 then print "a is 1" }

>

Compares two strings or numbers. Returns -1 if the first is greater than the second, 0 otherwise.


if a > 1 then print "a is greater than 1"

>=

Compares two strings or numbers. Returns -1 if the first is greater than or equal to the second, 0 otherwise.


if a >= 1 then print "a is greater than or equal to 1"

>>

A binary operator, shifts the left side rightt by the value in the right side. This is the same as divding by a power of 2 but it's much quicker


print 16 >> 3
would print 2

abs(n)

Returns the absolute value of the parameter


print abs(-4) , abs(5)
prints 4 5

and

Bitwise and of two numbers, which must be integers.

asc(string)

Returns the ASCII code of the first character of the string or zero if empty.


print asc("!")
prints 33

assert expr

Assert asserts a contract and is useful for parameter validation and testing. It evaluates the provided expression, and providing it is non zero. If it is zero, an error occurs.

An example use might be

assert name$<>""
to check that a name has been provided to a routine.

call

Call a named procedure. The brackets are required.

call draw.everything()

chr$(n)

Returns a one character string containing the character whose ASCII code is n


print chr$(42)
prints *

clear

Clear effectively resets the BASIC interpreter without actually restarting the program. It erases all variables and arrays, resets the stack and memory allocation, and checks for callable procedures. It should not normally be used as the RUN command does this as well. CLEAR takes an optional parameter which sets the high memory address for BASIC to use, so CLEAR $7E00 will mean any memory above $7E00 will not be used ; the stack and string will be placed there.

data (data)

Data for read statements. Note the string data must be enclosed in quotes.

data "Fred Smith",42,123456

dec(string)

Converts the string to an integer, assuming hexadecimal format


print dec("7ffe")
prints 32766

deek(n)

Returns the word at memory location n


print deek(32707)

dim

Dimensions an array. Arrays can have 1 or 2 dimensions. Multiply arrays can be declared in the same statement. Array Identifiers and procedures cannot have the same name (and if you think about it it's really not good practice to do so.)


dim name$(10),age(10): dim grid(8,8)

div

Binary operator, divides two numbers and returns an integer result. This is useful because BASIC is faster with integers.


print 11 div 4
will display 2

do .. loop .. exit

Repeats a block of code until you finish the loop by using the EXIT command

c = 0: do: c = c + 1:if c = 10:exit:endif:print c:loop

doke a,n

Writes the word a at address n

doke $0FF0,32766

end

Ends the current program and returns to the main prompt.

for .. next

Repeats a block of code a given number of times. The index variable (I in the examples) starts at the first value and counts towards the second. The step is 1 by default, but can be any number.

for i = 1 to 10:print i:next

for i = 1 to 20 step 2:print i:next

frac(n)

Returns the fractional part of the number, e.g. without the integer part.


print frac(-4.3) , frac(5.6)
prints 0.3 0.6

gosub...return

Subroutine call. Provided for backward compatibility ; please use PROC and CALL.

goto line

Transfer execution to given line number. Not recommended except for old software. Please use the structures provided.

hex$(n)

Converts the integer to a hexadecimal string equivalent


print hex$(154)
prints 9A

if (test) then (code)

Executes the rest of the line if the test is true, otherwise goes to the next line. Same as standard BASIC.

if n % 2 = 0 then print n;" is even"

if test :(code):else (code):endif

An extended multi line nestable if .. then ... else ... endif structure. The else clause is optional. An example is shown below ; this could be spread out over multiple lines. Note the absence of THEN ; this is the syntactic difference between the two.

if n % 2 = 0:print n;" is even":else:print n;"is odd":endif

inkey$()

Returns either an empty string if there is no keypress available, or a single character which is the next key in the keyboard queue.

print asc(inkey$())
prints 0 (or the ASCII code of the key)

input

Inputs values into variables. The syntax is the same as print, so semicolons, colons and quoted text works as normal.

INPUT "Enter your name ";a$

int(n)

Returns the integer part of the number, e.g. without the fractional part.


print int(-4.327) , int(5.61)
prints -4 5

left$(string,count)

Returns the leftmost 'count' characters of a string.


print left$("dennis",3)
prints den

len(string)

Returns the length of the string


print len("Hello")
prints 5

let

Assignment statement - sets a variable to a value. The LET is optional.


let count = 42 : name$ = "Agragjag"

list start,end

Program listing. Start and end are optional.

mid$(string,start<,count>)

Returns a substring of a string. start is the start position, which is indexed from 1. The count parameter, which is optional is the number of characters to extract.


print mid$("dennis",3)
prints nnis


print mid$("dennis",4,2)
prints ni

mod

Binary operator, returns the remainder of the integer division of two numbers.


print 11 mod 4
will display 3

new

Erase current program and variables and then return to command line.

or

Bitwise or of two numbers, which must be integers.

peek(n)

Returns the byte at memory location n


print peek(32707)

poke a,n

Writes the byte a at address n

poke $0FF4,42

print

Prints the following items on the display, can be a mixture of strings and numbers. A semicolon can be used to seperate elements, and a comma to move to the next tab position. If the last character is not a semicolon or comma, a new line is printed after this.


PRINT "Hello";name$,"tabbed" for example.

proc (name)() ... endproc

Defines a procedure. A procedure definition must be the first thing on the line.

proc bump.me():a = a+1:endproc

rand(number)

Returns a random integer between 0 and number-1.


print rand(4)
prints one of 0,1,2 or 3.

read (variables)

Read data from data statements into variables. The types must match

read name$,age,salary

rem

REM inserts comments in code. It is slightly different from most basics in that it requires the text of the REM statement to be in quotes. The tokeniser will do this for you, so if you type REM this is a comment it will convert it to REM "this is a comment"

It is purely informative ; there are two forms,

REM
without a comment and

REM "comment text"
with a comment.


REM "Calculate the players initial positions"

repeat .. until

Repeats a block of code until a condition is true.

c = 5:repeat:c = c-1:print c:until c = 0

restore

Reset the READ/DATA position to the start.

right$(string,count)

Returns the rightmost 'count' characters of a string.


print right$("dennis",3)
prints nis

rnd(1)

Returns a random number between 0 and 1. Always use the parameter 1 at present.


print rnd(1)
prints , say 0.6307

run

Runs the current program, clearing all variables and arrays first.

sgn(n)

Returns the sign of the number n ; 0 if zero, -1 if negative, 1 if positive and not zero


print sgn(-13)
prints -1

sqr(n)

Returns the square root of the given number


print sqr(144)
prints 12.0

stop

Stop the current program running and display an error message.

str$(n)

Converts the number to a string equivalent


print str$(154.9)
prints 154.9

sys a

Call machine code routine at a ; loads the A X and Y variables into the same named 6502 registers

sys $C008

val(string)

Converts string to a number. Errors if not possible.


print val("146")
prints 146

while .. wend

Repeats a block of code while a condition is true. The main difference between WHILE and REPEAT is that WHILE may never be executed, if the test is initially false.

x = 0:while x < 10:print x:x = x+1:wend

xor

Bitwise xor of two numbers, which must be integers.
